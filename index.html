配置云端同步

使用 Supabase 免费数据库实现多设备同步

📘 如何免费创建 Supabase 项目？（点击展开）

1. 访问 supabase.com → 用 GitHub 账号登录（免费）

2. 点击 「New project」 → 输入项目名（如 zzb-study）→ 设置数据库密码 → 选择 ap-southeast-1（新加坡，国内访问快）

3. 创建完成后 → 左侧菜单 「SQL Editor」 → 粘贴以下 SQL → 点击 Run：

-- 创建学习记录表
CREATE TABLE study_logs (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id) ON DELETE CASCADE NOT NULL,
  category TEXT NOT NULL CHECK (category IN ('computer', 'math')),
  content TEXT NOT NULL,
  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  timestamp TEXT NOT NULL
);

-- 开启行级安全
ALTER TABLE study_logs ENABLE ROW LEVEL SECURITY;

-- 用户只能读写自己的记录
CREATE POLICY "Users manage own logs" ON study_logs
  FOR ALL USING (auth.uid() = user_id)
  WITH CHECK (auth.uid() = user_id);

4. 左侧菜单 「Project Settings」 → 「API」 → 复制 Project URL 和 anon public key

5. 粘贴到下方输入框中 → 保存 → 即可开始同步 🎉
